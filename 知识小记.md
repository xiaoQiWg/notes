## JSON 方法混乱小记

    stringify: 字符串化
    parse: 解析
```js
  var json1 = '{"1":1,"2":2,"3":3}';
  var json2 = {'one':1,'two':2,'three':3};

  // 将字符串转换为对象        方法用来解析JSON字符串，构造由字符串描述的JavaScript值或对象。
  console.log(JSON.parse(json1));
  // 将对象转换为json数据      序列化成 一个JSON 字符串的值。
  console.log(JSON.stringify(json2)); 
```
